# ATHLETE OS

Mobile-first PWA for athlete readiness, tournament countdown, training logs, and calendar-based availability planning.

## Deploy on Netlify

1. Push this folder to a GitHub repository.
2. In Netlify, import the GitHub repository.
3. Use these settings:
   - Publish directory: `.`
   - Functions directory: `netlify/functions`
4. Add environment variables from `.env.example`.

## Mobile use

After deploying, open the Netlify URL on iPhone or Android and use Add to Home Screen / Install app.
